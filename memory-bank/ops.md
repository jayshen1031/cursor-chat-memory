3. 使用 Agent 模式
要使用 MCP 工具，需要确保在聊天中使用 Agent 模式，而不是普通的 Ask 模式。
4. 可用的 MCP 工具
一旦配置成功，您将可以使用以下工具：
sync_chat_data - 同步聊天数据
get_memory_bank - 获取记忆库内容
update_memory_bank - 更新记忆库
analyze_conversations - 分析对话
generate_summary - 生成摘要
